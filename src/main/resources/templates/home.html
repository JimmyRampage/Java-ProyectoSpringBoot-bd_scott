Proyecto Spring Boot con Base de Datos Scott

Esta aplicación es un ejemplo de un CRUD (Crear, Leer, Actualizar, Borrar) desarrollado con Spring Boot.

La aplicación gestiona la base de datos clásica de Scott, permitiendo realizar operaciones sobre las tablas de empleados (emp) y departamentos (dept).

Utiliza las siguientes tecnologías principales:

  • Spring Web: Para la creación de la aplicación web y los controladores REST.
  • Spring Data JPA: Para la persistencia y el acceso a los datos de forma sencilla.
  • Thymeleaf: Como motor de plantillas para generar las vistas HTML.
  • MySQL Connector/J: Para la conexión con la base de datos MySQL.
